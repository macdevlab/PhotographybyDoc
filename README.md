# Doc Photo Shop — Website Mockup

A homepage design mockup for **Doc Photo Shop**, photography by Sonya Samuel.

Services: Weddings, Portraits, and Events.

## About this mockup

This is a static HTML design concept built to show Sonya how her website could look and feel before committing to a platform. All photography currently shown is free stock imagery from [Unsplash](https://unsplash.com) used for demonstration — every image will be replaced with Sonya's own work once the portfolio is ready.

## Live preview

Once deployed to GitHub Pages, the site will be available at:
`https://<your-github-username>.github.io/doc-photo-shop/`

## What's included

- Sticky top navigation with enquire CTA
- Full-bleed hero with wedding photography background
- Three-service grid (Weddings, Portraits, Events) with "from" pricing
- Testimonial block
- About Sonya section
- Recent work gallery
- Dark CTA block with mailto enquiry button
- Responsive mobile layout
- Graceful fallback to warm gradients if any stock image fails to load

## Recommended next steps for Sonya

1. **Replace stock imagery with real photography** — 40–60 curated images across the three service categories
2. **Update testimonial** with a real quote and client attribution
3. **Write a full About page** (currently a teaser on the homepage)
4. **Decide on platform** for the live site — recommended: **Squarespace** (Farro template)
5. **Register the domain** `docphotoshop.com` or `docphotoshop.co.uk`

## Tech notes

- Single static `index.html` file — no build step required
- Fonts: Cormorant Garamond (serif) and Inter (sans) via Google Fonts
- Stock imagery: Unsplash (free to use under the Unsplash License)
- Responsive: works on mobile, tablet, desktop
- Minimal JavaScript — only for graceful image fallbacks
